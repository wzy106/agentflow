## 先搞清楚时间线

```
现在是 2026 年 5 月（大一下期末）
│
├── 2026.06-08  大一下暑假          ★ 黄金窗口1：算法 + 后端 + Agent 三线并行
├── 2026.09-12  大二上              ★ 项目深化 + 八股文

│
└── 目标：2027 年 3 月拿到日常实习 offer
```

> **为什么日常实习比暑期实习好进**：暑期实习（6-8月）是统一招聘，全国竞争；日常实习是部门缺人就招，全年滚动，竞争小很多。

## 大一大二生的核心策略：不要贪多

很多人大一大二疯狂学各种技术栈（Go、Rust、K8s、微服务），结果面试一道反转链表没写出来，直接挂。

**三个东西必须精，其他都可以糊弄**：
1. **算法**（一票否决）
2. **八股文**（聊得下去）
3. **一个硬核项目**（有的聊）

语言的边角料、Docker、CI/CD、消息队列——用到时半天就会，别现在花时间。

## 四阶段具体安排

### 第一阶段：三线并行（2026.06-08，暑假两个月）

暑假不要只啃算法——上午头脑最清醒时刷题，下午学后端写代码，晚上做项目。三线交叉不容易腻。

**每天节奏**：

```
08:30-11:30   算法（3h）   头脑清醒，攻克新题
11:30-14:00   午饭 + 午休 + 运动   别坐着不动
14:00-16:30   后端学习（2.5h）  看笔记 → 写代码 → 跑通
16:30-17:00   自由
17:00-19:00   晚饭 + 休息
19:00-21:00   Agent 项目（2h）  复刻/改进/写自己的
21:00-22:00   回顾今日 + 写笔记（1h）
22:00 之后    不看屏幕了
```

#### 算法线：LeetCode 100 题精刷（目标：中等题 25 分钟 bug-free）

减少到 100 题但每道题吃透——做过的不复习等于没做。**每天上午 2 道新题 + 1 道昨日复习**。

| 周 | 专题 | 题量 | 核心题（必做） |
|:---:|------|:---:|------|
| W1 | 数组 + 双指针 | 10 | 两数之和、三数之和、盛水最多容器、移动零、合并两个有序数组 |
| W2 | 链表 | 10 | 反转链表、环形链表、合并有序链表、删除倒数第N个、回文链表、相交链表 |
| W3 | 哈希表 + 字符串 | 10 | 字母异位词分组、最长无重复子串、字符串转整数、实现 strStr()、第一个唯一字符 |
| W4 | 栈 + 队列 | 8 | 有效括号、最小栈、用栈实现队列、每日温度、滑动窗口最大值 |
| W5 | 二叉树遍历 | 10 | 前中后序(递归+迭代)、层序遍历、锯齿形层序遍历 |
| W6 | 二叉树应用 | 8 | 最大深度、对称二叉树、路径总和、最近公共祖先、验证BST |
| W7 | 回溯 | 10 | 全排列、子集、组合总和、电话号码字母组合、括号生成 |
| W8 | 二分 + 贪心 | 8 | 二分查找、搜索旋转排序数组、寻找峰值、跳跃游戏、分发饼干 |
| W9 | 动态规划 I | 8 | 爬楼梯、打家劫舍、最大子序和、不同路径、最长递增子序列 |
| W10 | 动态规划 II | 8 | 零钱兑换、编辑距离、最长公共子序列、单词拆分 |
| W11 | 图 + 高频题 | 10 | 岛屿数量、课程表、克隆图 + LeetCode 上搜"字节跳动"高频前 6 |
| W12 | 复习 + 模拟 | 回顾 | 随机抽题，限时 25 分钟，手写（暑假检验） |

**检验标准**：第 12 周随机抽 10 道中等题，每道 25 分钟内写完，至少 8 道一遍过编译。

#### 后端线：从零到能写接口（同步走完速通笔记 Day 4-7）

跟着 `后端速通笔记.md` 走，重点是**每学一节就写代码，不只看**。

| 周 | 内容 | 产出 |
|:---:|------|------|
| W1-2 | HTTP 协议吃透 | 用 `requests` 调用 5 个公开 API（天气、翻译、搜索），每个都拿到正确结果 |
| W3-4 | FastAPI 入门 | 写一个独立服务：4 个 CRUD 接口 + Pydantic 模型 + Swagger 文档 |
| W5-6 | FastAPI 进阶 | 给上周的服务加：中间件日志 + Depends 认证 + 异常处理 |
| W7-8 | SQL + Redis | SQLite 存 3 张表做关联查询；Redis 实现 SET/GET/计数器 |
| W9-10 | 认证 + asyncio | 实现完整注册登录流程（JWT 签发和验证）；用 `asyncio.gather` 并行调 3 个 API |
| W11-12 | 整合 + 压测 | 把 FastAPI + SQLite + Redis + JWT 整合成一个完整后端服务，用 `httpx` 批量调接口看撑不撑得住 |

**检验标准**：第 12 周结束时，能不看教程**手写**一个 FastAPI 服务——包含用户注册登录、数据 CURD、缓存、限流。

#### Agent 线：从复刻到创新（持续交叉推进）

核心思路：**先跑通，再拆解，再改进**。不要闷头造轮子。

| 阶段 | 周 | 做什么 |
|:---:|:---:|------|
| 复刻 | W1-3 | **跑通 3 个开源 Agent 项目**（GitHub 搜 "agent framework" 或 "AI agent python"），每个都把 README 里的 demo 跑起来。理解它们的 tools 定义、消息循环机制 |
| 拆解 | W4-6 | **选一个最喜欢的项目读源码**。画出它的架构图：入口在哪、消息怎么流转、工具怎么注册和调用。把它简化重写——去掉花里胡哨的功能，只保留核心循环 |
| 改造 | W7-9 | **给简化版加功能**：① 加上对话历史存 SQLite ② 支持流式输出（`stream=True`）③ 加上错误处理（工具调用失败时重试或降级） |
| 创新 | W10-12 | **做一个"多工具智能助手"**：至少接入 3 个工具（计算器、网页搜索、天气查询），整合 FastAPI 变成 API 服务。这是你简历上第一个项目 |

**第 12 周产物**：一个能跑的 Agent 服务，`POST /chat` 发消息，Agent 自动决定调哪个工具，结果存数据库。

### 第二阶段：项目深化 + 八股文（2026.09-12，大二上）

学期中课程占时间，每周有效 ~20 小时。这个阶段重点从"学技术"转向"做项目"。

**学期节奏**：

```

  - 八股文：每天看 1 节小林 coding

周六日：每天 5-6h（集中做项目）

```

**9 月**：Agent 项目升级
- 把暑假的多工具助手用 FastAPI 重写（如果暑假已经 FastAPI 了就直接升级）
- 加 Docker 容器化，写 docker-compose.yml 一键启动
- 部署到云服务器（买阿里云/腾讯云学生机，99/年）

**10 月**：启动项目 2 —— RAG 知识库问答系统
- 文档上传解析（PDF/Word/TXT）
- 文档切片 + Embedding + pgvector 向量存储
- 用户提问 → 检索相关片段 → 拼 prompt → LLM 回答
- 回答溯源（来源标注）

**11 月**：两个项目收尾
- 两个项目的 README 都写到位（安装步骤、API 文档、架构图）
- 录一个 3 分钟演示视频（传到 B 站，放 README 里）
- GitHub 持续 commit，别断绿点

**12 月**：八股文第一遍 + 简历初稿
- 小林 coding 四门过完一遍
- 简历写出来，给学长/朋友看，改 3 版

**八股文（每周 5-8 小时，学期中慢慢看，不求背完）**：

先只看最核心的：

| 科目 | 先看什么 | 资源 |
|:---|------|------|
| 计算机网络 | HTTP/HTTPS、TCP 三次握手四次挥手、DNS | 小林 coding |
| 操作系统 | 进程线程协程、死锁、虚拟内存 | 小林 coding |

**方法**：不要死记背，每看完一章用自己的话把流程图重画一遍。面试本质是"你能不能用简单的话把复杂的东西讲清楚"。

### 第三阶段：冲刺（2027.01-02，寒假两个月）

这是面试前最后一个整块时间，优先级：

**优先级 1：刷题维持手感（每天 2 道）**

LeetCode 高频题查漏补缺，重点是写过但不熟的题重做。

**优先级 2：把项目吃透（关键！）**

面试官会深挖项目，你每个技术选型都要能回答"为什么"：

| 面试官问 | 你得能答 |
|:---|------|
| 为什么用 FastAPI 不用 Flask？ | 异步支持、自动生成文档、类型校验 |
| 为什么用 JWT 不用 Session？ | 无状态、不存服务端、适合分布式 |
| 为什么用 Redis 做限流而不是 MySQL？ | Redis 单线程 + 内存 + INCR 原子操作 |
| 数据库怎么设计的？为什么这样设计？ | 画出 ER 图，解释表关系和范式 |
| 怎么防止 API 被刷？ | Redis 计数器 + 滑动窗口 |
| 项目有什么难点？你怎么解决的？ | 准备一个具体的技术难点案例 |

**优先级 3：八股文系统背诵**

寒假集中过一遍所有八股，每看完一章就自己给自己讲一遍（录音，听自己讲的顺不顺）。

**优先级 4：写简历**

简历就一页。格式：

```
教育背景（3行）
- 福州大学 计算机/软件工程 大二  GPA 3.X/4.0

技术栈（1行）
- Python, C++, FastAPI, SQL, Redis, Docker, Git

项目经验（占简历 60%，核心部分）
### Agent 管理平台 | FastAPI + Redis + Docker
- 实现了多 Agent 创建、对话管理和 API 限流
- 使用 JWT 实现用户认证，Redis 实现接口级限流
- 对话历史使用 SQLite 持久化，支持按 session 查询
- 部署到阿里云，通过 Docker Compose 一键启动
  链接：https://github.com/xxx/agent-platform

### RAG 知识库问答系统 | FastAPI + PostgreSQL + pgvector
- 实现文档上传、向量化、语义检索和 LLM 问答
- 支持 PDF/Word/TXT 多种格式解析
- 答案溯源，标注来源文档和段落
  链接：https://github.com/xxx/rag-qa

个人能力（2-3行）
- LeetCode 200+ 题，周赛最好 xxx 分
- 熟悉常用数据结构与算法，中等难度题目 20 分钟 bug-free
```

**面试话术**：

Q: "你是大二的学生？"
A: "是的，我大二。我大一暑假开始系统学后端，做过两个独立项目并部署上线。算法准备了 200 多道题。"

## 别踩的坑

1. **收藏即学会**——B 站教程收藏 50 个，一行代码没写。**解决**：看一节写一节。
2. **完美主义**——想等"准备好"再投简历。**永远没有准备好的时候**，寒假做完了就投。
3. **同时学太多语言**——又想学 Go 又想学 Java。**大二之前只用 Python**，学透一门再换。
4. **忽略算法**——觉得写项目更爽就一直写项目。**算法是大厂一票否决**，先过算法关再搞项目。
5. **只刷题不复习**——做一道忘一道。**复习比做新题重要**，每天花一半时间复习昨天的。
6. **项目只有代码没有文档**——面试官打开 GitHub 不知道怎么跑。**README 写清楚：怎么安装、怎么运行、有哪些接口**。
